This directory contains the source of FACTORIE, a toolkit for
probabilistic modeling based on imperatively-defined factor graphs.

More information is available at http://code.google.com/p/factorie.

Installation currently relies on Maven, version 2.  If you don't
already have maven, version 2, install it from
http://maven.apache.org/download.html.  Then type "mvn compile".
